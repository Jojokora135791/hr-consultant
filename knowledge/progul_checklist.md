# Чеклист: Прогул — отсутствие на рабочем месте без уважительной причины

> Источник: внутренняя инструкция Контур. Обновлять этот файл при изменении регламентов.

## Квалификация

Отсутствие работника на рабочем месте без уважительной причины или без согласования с непосредственным руководителем — дисциплинарное нарушение.

**Важно:** во время отсутствия не засчитывается обеденное время. При длительном прогуле исключаются: дни на больничном, в согласованном отпуске, выходные, праздники, дни командировки.

## Порядок действий

### Шаг 1. Свяжись с работником для установления причины
- Позвони по известному номеру телефона (в т.ч. из корпоративного справочника Стафф)
- Если не выходит на связь — позвони родственникам по номерам из анкеты при трудоустройстве
- Сделай запрос на корпоративную электронную почту работника
- Используй соцсети, чтобы понять причину невыхода

### Шаг 2. Зафиксируй неявку
- Составь **акт об отсутствии на рабочем месте** в конце рабочего дня
- В акте укажи ФИО трёх работников-свидетелей, подтверждающих отсутствие
- Если работник не вышел на следующий день — повтори процедуру (каждый день отсутствия фиксируется отдельно)

### Шаг 3. Передай информацию в отдел кадрового учёта
- Сообщи Касмыниной Ольге или Черепановой Татьяне
- В табеле учёта рабочего времени будет проставлено: **НН** (отсутствие по невыясненным причинам)

### Шаг 4. Инициируй служебную проверку
Подготовь **служебную записку**. Обязательные элементы:
- ФИО и должность/подразделение работника
- Дата и время отсутствия
- Предпринятые меры (звонок работнику с выяснением причины — с подтверждающими документами)
- Обстоятельства отсутствия (установлена ли причина)
- Отсутствие информации по проактивным выплатам из Экстерн (нет больничного листа)
- Предложение о вынесении дисциплинарного взыскания

### Шаг 5. Ожидай результаты служебной проверки
- Ответственное лицо подготовит Требование о предоставлении объяснения
- Направит работнику заказным письмом по адресу проживания
- По результатам — решение о применении дисциплинарного взыскания

## Нормативная основа
- ТК РФ ст. 81 ч. 1 п. 6 пп. «а» — прогул как основание для увольнения
- ТК РФ ст. 193 — порядок применения дисциплинарного взыскания
- ПВТР Контура п. 13.2.1 — режим работы и рабочее место
