## Инструкции для LLM по заполнению

- {{адресат_фио}}, {{адресат_должность}} — эксперт по трудовому законодательству (Черепанова Т. / Касмынина О.)
- {{нормативная_ссылка}} — заполнить из rag_violations (ТК РФ ст.81 п.6 пп.а, ПВТР, трудовой договор)
- {{обстоятельства_отсутствия}} — из контекста: удалось ли выйти на связь, есть ли объяснение
- {{описание_мер}} — из контекста: что предпринял руководитель (звонок, письмо и т.д.)
- {{фио_работника_род}} — родительный падеж от ФИО (Иванова И.И. → Иванова И.И.)
- {{дополнительные_приложения}} — если есть скрины/доказательства из контекста
- Если данных не хватает — заполнить [УТОЧНИТЬ]
